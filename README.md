# PoWARE⚡

PoWARE is a handy breadboard power supply for the modern maker & engineer. 
 - 🔌 USB-C PD powered
 - ⚙️ Adjustable rail voltages: 3V3–5V
 - 🪛 Screw terminal for direct PD access: 5–20V
